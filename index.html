How does it work?

(Or more like wut?)

1. Install rebot

2. Misconfigure any service

3. Start that service

4. Service shoots logs

5. Rebot detects mistakes using logs

6. Rebot looks for answers on web

7. Rebot fixes your mistake

Queries?

1. Your personal Google will not fit in a traditional relational database.

2. Logs are also massive. (Papertrail, loggly, ..)

(What will my queries be?)

1. Search massive database of web for answers.

2. Find a union of solutions with information from logs.
